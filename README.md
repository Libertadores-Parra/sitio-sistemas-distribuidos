# sitio-sistemas-distribuidos
 Sitio web sobre Sistemas Distribuidos - Actividad práctica


 1. ¿Qué ventajas encuentras en trabajar directamente en la nube con GitHub?

Trabajar en la nube con GitHub tiene varias ventajas importantes. Permite acceder al proyecto desde cualquier lugar sin necesidad de instalar software adicional, lo que facilita mucho la movilidad. Además, el control de versiones es automático, por lo que puedes ver cambios, regresar a versiones anteriores y trabajar en equipo sin conflictos graves. También simplifica la colaboración, ya que varias personas pueden contribuir al mismo proyecto en tiempo real. Finalmente, la integración con servicios como GitHub Pages permite desplegar sitios web de manera rápida y gratuita.

2. ¿Qué desafíos tuviste al crear el sitio sin usar herramientas locales?

Uno de los principales desafíos es la falta de un entorno de desarrollo completo, ya que no siempre se cuenta con herramientas avanzadas como editores con autocompletado o depuración en tiempo real. También puede ser más difícil detectar errores rápidamente. Otro problema es la dependencia de la conexión a internet, ya que sin ella no se puede trabajar. Además, algunas tareas simples pueden volverse más lentas al no tener control total del entorno como en una máquina local.

3. ¿Cómo podrías mejorar este sitio web en el futuro?

El sitio se puede mejorar agregando interactividad con JavaScript, como animaciones o validación de formularios. También se podría hacer más responsivo para adaptarse mejor a dispositivos móviles. Otra mejora sería integrar un backend para manejar datos reales, como formularios funcionales. Además, se podría optimizar el diseño visual con mejores componentes, iconos y efectos modernos. Finalmente, implementar buenas prácticas como accesibilidad y optimización SEO haría el sitio más profesional.

4. ¿Qué aprendiste sobre Sistemas Distribuidos al crear este sitio?

Aprendí que los sistemas distribuidos permiten que múltiples recursos trabajen juntos como si fueran uno solo, incluso estando en diferentes ubicaciones. GitHub es un claro ejemplo, ya que permite almacenar, versionar y compartir código en servidores distribuidos. También comprendí la importancia de la sincronización, la disponibilidad y la tolerancia a fallos. Este proyecto ayudó a entender cómo la nube facilita el desarrollo y despliegue de aplicaciones de forma eficiente y colaborativa.
